# ExpenseTracker
Live Demo — [Ознакомиться](https://bobicentr.github.io/expensetracker)

# О проекте
Приложение для учёта личных расходов.  
Позволяет добавлять, редактировать и удалять записи о тратах, при этом все данные сохраняются в LocalStorage и остаются при перезагрузке страницы. 

# Технологии
- HTML, CSS, Bootstrap — верстка и адаптивный интерфейс  
- Vanilla JS (ES6+) — логика приложения, работа с DOM и событиями  
- LocalStorage — хранение записей и суммарных данных  

# Как запустить
1. Склонировать репозиторий
```bash
git clone https://github.com/bobicentr/expensetracker.git
cd expensetracker
```

2. Открыть проект через локальный сервер (например, Live Server)  

# Что умеет
- Добавление новых расходов с указанием категории, суммы и даты  
- Редактирование и удаление существующих записей  
- Автоматическое сохранение всех данных в LocalStorage  
- Подсчёт общей суммы расходов в реальном времени  
- Адаптивный интерфейс, удобный на мобильных устройствах
